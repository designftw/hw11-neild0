Exercise 1: Adding Animations to the Chat App

Animation 1: Movement for other people's messages

To make new messages less jarring, other people's messages glide in the from the left.

Animation 2: Movement for your own messages

To make your own messages more noticeable, your messages glide in from the right.

Animation 3: Movement for the send text

When the user clicks send, it also gently pushes the send text up, to make it clear that the message is being sent.

Animation 4: Transtion between like and unlike

When the user clicks like, it shrinks down and expands back up with the dislike text. When the user clicks dislike, it shrinks down and expands back up with the like text. This is to make it clear to the user that their action has been registered.

Animation 5: Fade out for deleted messages

When the user clicks delete, the message fades out. This is to make it clear to the user that the message has been deleted, without being jarring.
